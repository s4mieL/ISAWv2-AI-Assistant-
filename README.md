# ISAW v2 - AI Assistant

**ISAW v2** is the second iteration of my AI assistant project, focused on leveraging large language models (LLMs) to provide intelligent conversation and task assistance. This project is **ongoing**, exploring AI-driven interactions while I continue to expand its capabilities in future versions.

---

## Features
- Intelligent conversation powered by LLMs
- Task assistance and automation (planned for future updates)
- Modular design for easy expansion and integration
- Lightweight and easy to deploy locally

---

## Installation
1. Clone the repository:  
```bash
git clone https://github.com/your-username/ISAWv2-AI-Assistant.git
```
2. Navigate to the project directory:
```bash
cd ISAWv2-AI-Assistant
```
3. Install dependencies:
```bash
pip install -r requirements.txt
```
4. Run the Assistant
```bash
python open_ai_testv3.py
```
**Usage**

Start a conversation with the AI assistant by running main.py.

Commands and task functionalities will be gradually expanded in future updates.
